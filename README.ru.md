# YT Lang Learning

[English](README.md) | Русский

Превратите YouTube-ролики в учебные материалы. YT Lang Learning показывает видео, английские субтитры и русский перевод в одном окне, поэтому не приходится переключаться между плеером и переводчиком.

![Учебный плеер YT Lang Learning с английскими и русскими субтитрами](Docs/screens/study-player.png)

## Возможности

- Собирайте YouTube-уроки в папки и меняйте их порядок по мере обучения.
- Нажимайте на субтитр для перехода к реплике, повторяйте текущую фразу, перематывайте на пять секунд и меняйте скорость.
- Загружайте доступные английские субтитры, создавайте русский перевод через OpenRouter или распознавайте речь локально через faster-whisper, когда у YouTube нет английской дорожки.

Приложение запоминает позицию каждого YouTube-ролика. Библиотека, настройки и файлы субтитров остаются на компьютере.

## Установка и запуск

Сборка проекта создаёт установщик `yt-lang-learning-setup.exe` и ZIP-архив.

### Установщик

1. Запустите `yt-lang-learning-setup.exe`.
2. Завершите установку и откройте YT Lang Learning.

### ZIP-архив

1. Распакуйте весь архив в отдельную папку.
2. Запустите `yt-lang-learning.exe` из этой папки.

Не переносите исполняемый файл отдельно от остальных файлов сборки. Инструкции для запуска и сборки из исходников находятся в [технической документации](Docs/README.md).

## Первый урок

1. Откройте раздел **БИБЛИОТЕКА**.
2. Нажмите **＋ РОЛИК**, вставьте полную ссылку YouTube и укажите название урока.
3. Дважды нажмите на сохранённый ролик, чтобы открыть плеер.
4. Нажмите **ЗАГРУЗИТЬ EN** и подтвердите запрос к YouTube.
5. Если нужен русский перевод, сохраните API-ключ OpenRouter в настройках и нажмите **ПЕРЕВЕСТИ RU**.

Ролик можно сначала открыть без сохранения. Вставьте ссылку прямо в разделе **ПЛЕЕР**, а затем добавьте подходящий материал в библиотеку.

![Библиотека YT Lang Learning](Docs/screens/library.png)

## Режимы просмотра

**КОЛОНКИ** размещает английские субтитры слева, а русский перевод справа. Текущая реплика подсвечивается в обеих панелях.

**ЦЕНТР** показывает текущую английскую фразу и перевод рядом с видео. Кнопки EN и RU скрывают боковые панели, когда плееру нужно больше места.

## Перевод и распознавание речи

Для русского перевода нужен API-ключ OpenRouter. Приложение отправляет выбранной модели текст английских субтитров. Само видео для перевода не передаётся. Windows сохраняет ключ только при доступном системном шифровании.

Если YouTube не отдаёт английскую дорожку, приложение может предложить локальное распознавание речи. Для него нужны скачанная модель faster-whisper, `uv` и `yt-dlp`. Временное видео загружается только после подтверждения и удаляется после обработки.

![Настройки YT Lang Learning](Docs/screens/settings.png)

## Ограничения

- Для YouTube-плеера и загрузки субтитров нужен интернет.
- Ролик может не открыться, если его владелец запретил встраивание.
- Доступность английских субтитров зависит от YouTube. Для локального распознавания нужны отдельные инструменты и скачанная модель.
- Для русского перевода нужны аккаунт OpenRouter, API-ключ и доступ к выбранной модели.
