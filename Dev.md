Идея состоит в том, чтобы написать приложения для удобного изучения языка, в центре будет youtube-ролик проиграться. С левой и справа в колонках будут размещены субтитры по-английски (слева), справа по русски, либо можно переместить их по центру основной. Кнопочки внизу на ускорение видео для удобного изучения языка.
