# Руководство пользователя YT Lang Learning

Видео на изучаемом языке легко превращается в фоновый шум: знакомые слова слышны, а смысл целой реплики ускользает. YT Lang Learning разбирает YouTube-ролики по фразам и показывает исходную дорожку рядом с переводом.

Пригодится, чтобы:

- собирать учебные ролики в папки и возвращаться к ним позже;
- видеть реплику на изучаемом языке рядом с выбранным переводом;
- повторять сложные места, менять скорость и переходить к нужной фразе нажатием на субтитр.

## Что нужно для работы

Для просмотра роликов и загрузки субтитров нужен интернет. Приложение сначала запрашивает выбранную дорожку или машинный перевод у YouTube. При отсутствии дорожки либо ответе 429 оно предлагает OpenRouter или локальный Whisper.

Для Whisper заранее нужны модель, `uv` и `yt-dlp`; для локальных аудиофайлов также нужен `ffmpeg`. Решение о машинном переводе и резервном способе сохраняется отдельно для каждого ролика.

Библиотека, настройки и субтитры хранятся локально. В установленной сборке приложение хранит их в `%LOCALAPPDATA%/YT Lang Learning`.
