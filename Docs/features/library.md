# Библиотека учебных роликов

Библиотека хранит ссылки на уроки в привычном дереве папок. Так можно разделить длинный список видео по темам и быстро вернуться к нужному материалу.

![Библиотека YT Lang Learning](../screens/library.png)

## Основные действия

1. Нажмите кнопку с папкой рядом с заголовком «Библиотека», чтобы создать тему.
2. Выберите папку и добавьте в неё ролик кнопкой с изображением видео.
3. Перетаскивайте ролики и папки по дереву, чтобы изменить порядок обучения.
4. Нажмите правой кнопкой на элемент, чтобы добавить вложенный материал, переименовать его или удалить.
5. Дважды нажмите на ролик, чтобы перейти к просмотру.

Одинарное нажатие открывает свойства выбранного ролика. Там можно изменить название или ссылку, сохранить правки и открыть плеер.

Удаление требует отдельного подтверждения. Оно убирает запись из библиотеки, поэтому перед подтверждением проверьте выбранное название.
