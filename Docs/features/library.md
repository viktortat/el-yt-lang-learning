# Бібліотека навчальних роликів

Бібліотека зберігає посилання на уроки в дереві папок. Можна розділити відео за темами й швидко повернутися до потрібного матеріалу.

![Бібліотека YT Lang Learning](../screens/library.png)

## Кілька бібліотек

Кожна бібліотека має власну мовну пару. Наприклад, «Англійська», «Німецька» та «Польська» — три бібліотеки з різними налаштуваннями.

Щоб створити нову бібліотеку, натисніть на назву поточної й оберіть **Керування бібліотеками**. Там само можна перейменувати, видалити або експортувати будь-яку з них.

Мовна пара кожної бібліотеки налаштовується окремо й діє на всі ролики всередині неї за замовчуванням. Для конкретного відео її можна змінити в плеєрі.

## Основні дії

1. Натисніть кнопку з папкою біля заголовка «Бібліотека», щоб створити тему.
2. Виберіть папку й додайте ролик кнопкою з відео.
3. Перетягуйте ролики й папки деревом — порядок зміниться.
4. Клацніть правою кнопкою на елемент: меню дозволяє додати папку чи ролик, перейменувати або видалити.
5. Двічі клацніть на ролик, щоб відкрити плеєр.

Одинарне натискання відкриває властивості ролика — можна змінити назву або посилання.

Видалення потребує підтвердження й прибирає запис із бібліотеки.

## Імпорт із плейлиста YouTube

У властивостях папки є поле «Посилання на плейлист YouTube». Вставте URL і натисніть **Додати**. Застосунок знайде всі ролики, пропустить уже додані й запитає перед завантаженням нових.

## Імпорт та експорт бібліотеки

**Імпорт** замінює поточну бібліотеку на іншу з JSON-файлу. Перед заміною створюється резервна копія. **Експорт** зберігає бібліотеку в JSON — можна перенести на інший комп'ютер.

**Відновити** повертає останню автоматичну резервну копію.
