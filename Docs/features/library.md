# Библиотека учебных роликов

Библиотека хранит ссылки на уроки в дереве папок. Можно разделить видео по темам и быстро вернуться к нужному материалу.

![Библиотека YT Lang Learning](../screens/library.png)

## Основные действия

1. Нажмите кнопку с папкой у заголовка «Библиотека», чтобы создать тему.
2. Выберите папку и добавьте ролик кнопкой с видео.
3. Перетаскивайте ролики и папки по дереву — порядок изменится.
4. Кликните правой кнопкой на элемент: меню позволяет добавить папку или ролик, переименовать или удалить.
5. Дважды кликните на ролик, чтобы открыть плеер.

Одинарное нажатие открывает свойства ролика — можно изменить название или ссылку.

Удаление требует подтверждения и убирает запись из библиотеки.

## Импорт из плейлиста YouTube

В свойствах папки есть поле «Ссылка на плейлист YouTube». Вставьте URL и нажмите **Добавить**. Приложение найдёт все ролики, пропустит уже добавленные и спросит перед загрузкой новых.

## Импорт и экспорт библиотеки

**Импорт** заменяет текущую библиотеку на другую из JSON-файла. Перед заменой создаётся резервная копия. **Экспорт** сохраняет библиотеку в JSON — можно перенести на другой компьютер.

**Восстановить** возвращает последнюю автоматическую резервную копию.
