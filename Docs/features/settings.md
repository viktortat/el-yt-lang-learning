# Настройки

Здесь выбирается тема интерфейса, модель перевода и способ локального распознавания английской речи.

![Настройки YT Lang Learning](../screens/settings.png)

## Внешний вид

Выберите светлую или тёмную тему и нажмите «Сохранить настройки».

## Перевод через OpenRouter

1. Вставьте API-ключ OpenRouter в поле «API-ключ».
2. Проверьте имя модели перевода.
3. Нажмите «Сохранить настройки».

После сохранения приложение не показывает ключ в открытом виде. В Windows он записывается только при доступном системном шифровании. Если шифрование недоступно, приложение сообщит об ошибке и не сохранит ключ.

Когда ключ уже сохранён, пустое поле не удаляет его. Оно означает «оставить прежний ключ».

## Локальная транскрибация

Локальное распознавание нужно только для роликов без доступной английской дорожки YouTube. Укажите:

- каталог с уже скачанной моделью faster-whisper;
- название модели;
- команды запуска `uv` и `yt-dlp`.

Приложение запускает локальную транскрибацию после отдельного подтверждения. Временное видео удаляется после обработки.

Кнопка «Сбросить» возвращает стандартные значения в открытую форму. Они попадут в файл настроек только после нажатия «Сохранить настройки».

Вернуться к [первому запуску](../getting-started.md) или [учебному плееру](study-player.md).
