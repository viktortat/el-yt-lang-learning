# Настройки

Здесь выбирается тема интерфейса, модель перевода и способ локального распознавания английской речи.

![Настройки YT Lang Learning](../screens/settings.png)

## Внешний вид

Выберите светлую или тёмную тему и нажмите «Сохранить настройки».

## Перевод через OpenRouter

1. Вставьте API-ключ OpenRouter в поле «API-ключ».
2. Проверьте имя модели перевода.
3. Нажмите «Сохранить настройки».

После сохранения приложение не показывает ключ в открытом виде. Если ключ уже сохранён, пустое поле не удаляет его — оно означает «оставить прежний». В Windows ключ записывается только при доступном системном шифровании. Если шифрование недоступно, приложение сообщит об ошибке.

## Локальная транскрибация

Локальное распознавание нужно только для роликов без английской дорожки YouTube. Укажите:

- каталог с уже скачанной моделью faster-whisper;
- название модели;
- команды запуска `uv` и `yt-dlp`.

Приложение запускает транскрибацию после вашего подтверждения. Временное видео удаляется после обработки.

Кнопка «Сбросить» возвращает значения по умолчанию в форму. В файл настроек они попадут только после «Сохранить настройки».

Вернуться к [первому запуску](../getting-started.md) или [учебному плееру](study-player.md).
